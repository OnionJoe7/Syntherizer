To set the system up for use locally, follow this guide

Apache server and MySQL needs to be setup:

1. Verify md5sum of xampp-windows-x64-8.1.4-1-VS16-installer.exe
2. Run xampp-windows-x64-8.1.4-1-VS16-installer.exe from ./dependencies
3. Run QuickSetup.py to setup database
4. Start apache and MySQL from control panel
5. Run Syntherizer_main.html

Full application is found inside the 'app' directory.
Whenever a password is entered incorrectly, it will be logged to a CSV in a suitable format inside the logging directory
